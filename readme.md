# HDSP-21xx-driver

A driver for LED character displays.

### Compatible Models

- HDSP-2107
- HDSP-2112
- HDSP-2110
- HDSP-2111
- HDSP-2113
- HDSP-2504 *
- HDSP-2502 *
- HDSP-2500 *
- HDSP-2501 *
- HDSP-2503 *

\* Due to their dimensions they are only software compatible and do not fit on the PCB.
